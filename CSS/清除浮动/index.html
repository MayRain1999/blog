CSS 清除浮动

高度坍塌是指父元素本来应该包括子元素的高度，但是实际上父元素比子元素的高度要小

父级添加 overflow: hidden​

<ul class="cc">
  <li></li>
  <li></li>
</ul>
<style type="text/css">
  li {
    list-style: none;
    height: 100px;
    width: 100px;
    float: left;
    background: red;
    margin-left: 20px;
  }

  ul {
    overflow: hidden;
    padding: 0;
    margin: 0;
    background: pink;
  }
</style>

clear:both​

<style type="text/css">
  li {
    list-style: none;
    height: 100px;
    width: 100px;
    float: left;
    background: red;
    margin-left: 20px;
  }
  ul {
    background: pink;
  }
</style>
<ul class="cc">
  <li></li>
  <li></li>
  <div style="clear: both;"></div>
</ul>

伪类 after​

<style type="text/css">
  li {
    list-style: none;
    height: 100px;
    width: 100px;
    float: left;
    background: red;
    margin-left: 20px;
  }

  .cc:after {
    content: "";
    height: 0;
    line-height: 0;
    display: block;
    visibility: hidden;
    clear: both;
  }
  ul {
    background: pink;
  }
</style>
<ul class="cc">
  <li></li>
  <li></li>
</ul>

双伪类​

<style type="text/css">
  li {
    list-style: none;
    height: 100px;
    width: 100px;
    float: left;
    background: red;
    margin-left: 20px;
  }

  .cc:after,
  .cc:before {
    content: "";
    display: block;
    clear: both;
  }

  ul {
    background: pink;
  }
</style>
<ul class="cc">
  <li></li>
  <li></li>
</ul>
